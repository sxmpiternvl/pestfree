# @monorepo/bot — Telegram-бот

Бот для приёма заявок через Telegram. Проводит пользователя через wizard-сцену, собирает данные и отправляет их в API. После успешной отправки уведомляет владельца.

## Стек

- **Telegraf 4** — фреймворк для Telegram-ботов
- **Telegraf Scenes / WizardScene** — пошаговый диалог
- **ts-node-dev** — hot-reload в dev-режиме

## Структура

```
apps/bot/
├── SETUP.md          — инструкция по созданию бота в BotFather
└── src/
    ├── main.ts       — точка входа, запускает бота (polling)
    ├── bot.ts        — создание экземпляра Telegraf, регистрация команд
    ├── types.ts      — BotContext с типизацией session
    └── scenes/
        └── lead.scene.ts  — WizardScene для сбора заявки
```

## Сценарий диалога

```
/start или кнопка "Оставить заявку"
        ↓
[Шаг 0] Выбор услуги (inline-кнопки)
        ↓
[Шаг 1] Обработка выбора услуги
        ↓
[Шаг 2] Ввод ФИО
        ↓
[Шаг 3] Ввод телефона (валидация: минимум 7 цифр)
        ↓
[Шаг 4] Ввод адреса (минимум 5 символов)
        ↓
[Шаг 5] Комментарий или «Пропустить»
        ↓
[Шаг 6] Подтверждение → POST /leads → уведомление владельца
```

## Команды бота

| Команда    | Описание                        |
|------------|---------------------------------|
| `/start`   | Приветствие + кнопка заявки     |
| `/request` | Войти в сцену напрямую          |
| `/cancel`  | Отменить текущую заявку         |

## Переменные окружения

| Переменная     | Обязательно | Описание                                      |
|----------------|-------------|-----------------------------------------------|
| `BOT_TOKEN`    | ✅           | Токен от @BotFather                           |
| `BOT_USERNAME` | ✅           | Username бота без `@`, напр. `pestfree_bot`   |
| `OWNER_CHAT_ID`| ✅           | Telegram ID владельца для уведомлений         |
| `API_BASE_URL` | ✅           | URL API, напр. `http://localhost:3001`        |

Подробная инструкция по получению токена и chat ID — в [SETUP.md](SETUP.md).
